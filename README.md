# ICSI-problem
Репозиторий с кодом для диплома по обработке видеозаписей с процедурой оплодотворения

### 1. Требования и библиотеки

В работе использовалось виртуальное окружение с установленным `Python 3.14`.

Полный список библиотек указан в файле `requirements-full.txt`, в том числе, которые подгружаются совместно с основными библиотеками:

```bash
python -m pip install -r requirements-full.txt
```

Список основных библиотек указан в файле `requirements.txt`:

```bash
python -m pip install -r requirements.txt
```

Обучение производилось на `NVIDIA GeForce RTX 4060 Laptop GPU`.
