# ICSI-problem

Репозиторий с кодом для диплома по обработке видеозаписей с процедурой оплодотворения

## 1. Требования и библиотеки

В работе использовалось виртуальное окружение с установленным `Python 3.14`.

Полный список библиотек указан в файле `requirements-full.txt`, в том числе, которые подгружаются совместно с основными библиотеками:

```bash
python -m pip install -r requirements-full.txt
```

Список основных библиотек указан в файле `requirements.txt`:

```bash
python -m pip install -r requirements.txt
```

Обучение производилось на `NVIDIA GeForce RTX 4060 Laptop GPU`.

## 2. Структура каталога

В репозитории хранятся, как и полезные файлы, так и промежуточные результаты исследования. Описание структуры соответствует логической схеме работы.

1. Определение наличия ооцита в кадре.
2. Определение границ объектов на изображении.
3. Анализ расхождения результатов.
4. Выделение ключевых признаков/объектов.
5. Прогнозирование исхода.

### 2.1 lines

Определение наличия ооцита и выявление границ объектов в кадре на основе контурно-геометрических методов. Рассматривалось несколько вариаций на основе opencv canny и приведение к полярным координатам с целью получить более достоверную границу объекта.

### 2.2 detection_metric

Подсчет метрик для выявления ооцита через контурно-геометрический метод и для нейронки ResNet18, обученной на датасете с пустыми изображениями и с ооцитом в кадре.

### segmentation

Обучение и получение весов для Unet++ для изображений приводимых к формату 512x512 и 768x768.

### texture_check

В данной папке содержатся файлы предварительной обработки изображения:

- извлечение пар "до" и "после" введения иглы
- кадрирование пар изображений по ооциту
- извлечение из пар до и после математических признаков изменения структуры ооцита

### analysis_changes_tenserflow

`Python 3.14` не поддерживал `tenserflow` с нужными библиотеками из-за чего в отдельном окружении на `Python 3.10` запускались данные файлы с установленными библиотеками `pandas`, `numpy`, `scikit-learn`, `tensorflow`.

Основная функция данного блока была в том, чтобы с помощью классических методов машинного обучения в `analysis_changes.ipynb` обучиться прогнозировать исход на основе математических признаков, извлеченных в `texture_check\texture_check_all.py`. В данные признаки входили:

- среднее изменение интенсивности между кадрами во внутренней области ооцита,
- максимальное изменение интенсивности между кадрами,
- доля пикселей внутренней области ооцита, для которых разница яркости между кадрами превышает порог 20,
- изменение средней яркости во внутренней области ооцита,
- изменение стандартного отклонения яркости,
- изменение энтропии изображения во внутренней области ооцита,
- изменение дисперсии лапласиана.
